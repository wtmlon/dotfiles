bash + tmux + vim

## Installation

### Linux

```bash
curl https://raw.githubusercontent.com/wtmlon/dotfiles/master/bootstrap.sh | bash
```

### Mac:

```bash
wget -O - https://raw.githubusercontent.com/wtmlon/dotfiles/master/bootstrap.sh | bash
```
